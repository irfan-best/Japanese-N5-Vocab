Keyboard Shortcuts & Help Guide

Keyboard Shortcuts

Efficiently navigate and manage your learning sessions using these dedicated shortcuts.

Shortcut Action
Ctrl + M Toggle Selection Mode (needed to use selection commands).
M (in Selection Mode) Move selected words to Hard (Normal list) / Move to Normal (Hard list).
D (in Selection Mode) Delete all selected words.
→ (Right Arrow) (in Normal Mode) Move focus to next card and play Japanese audio.
← (Left Arrow) (in Normal Mode) Move focus to previous card and play Japanese audio.
Space Repeat speaking current card's Japanese audio (or Quiz word).
Ctrl + Click Select only the clicked word (only in Selection Mode).
Shift + Click Select range of cards (only in Selection Mode).
Esc Close active modal dialog or clear card selection.
Ctrl + ← (Left Arrow) Go to the previous lesson (only when Selection Mode is inactive).
Ctrl + → (Right Arrow) Go to the next lesson (only when Selection Mode is inactive).
← / ↑ (Left/Up Arrow) (in Selection Mode) Move selected card(s) Up (left).
→ / ↓ (Right/Down Arrow) (in Selection Mode) Move selected card(s) Down (right).
Delete / Backspace Delete selected card(s).
Q Trigger / Close the Quiz modal.
I Open Import vocabulary modal and focus the text box.
C Copy N5 configuration/backup to clipboard.
T Toggle between Light and Dark themes.
Shift + 1 .. 5 Change display modes (1: Big English, 2: Big Jap, 3: Eng, 4: Jap, 5: Romaji).
Enter (in Quiz Setup) Start Quiz.
Enter (in Quiz Active) Reveal answer, submit text input, or go to next question.
← (Left Arrow) (in Quiz Active) Go to previous question (when text input is not focused).
→ (Right Arrow) (in Quiz Active) Go to next question (when text input is not focused).
Space (in Quiz Active) Read current word in Japanese (when text input is not focused).
↑ (Up Arrow) (in Quiz Setup) Select previous Quiz Mode.
↓ (Down Arrow) (in Quiz Setup) Select next Quiz Mode.
O (in Quiz Setup) Select Original question order.
R (in Quiz Setup) Select Random question order.
Ctrl + Enter (in Quiz Setup) Toggle between Normal and Hard mode.

Application Guidelines

Vocabulary Storage & Import

Words are organized inside words.js within an allWords dictionary. You can import new words by tapping the Import button. Use three-line blocks:

けんきゅうしゃ
researcher, scholar
kenkyuusha

Backup Workflow

  1. Configure your lessons, flag counts, and preferences.
  2. Click Copy to Clipboard to copy the output representation.
  3. Paste the output into words.js on your machine.
  4. Click Reset Cache to clean up local overrides; your past states are preserved in the code file.

Mobile Touch Controls

Learn how to navigate and manage your vocabulary using simple gestures on your mobile device:

Control Action
Tap Card (in Grid) Focus on a card and play its Japanese pronunciation.
Tap Card (in Quiz) Acts exactly like the Enter key (Reveal Answer or Next Question).
Selection Mode Toggle Toggle Selection Mode via the button next to "Copy to Clipboard" at the bottom of the page.
Tap Card (in Selection Mode) Directly toggles the word's selection (no Ctrl key required on mobile).
Search Bar Type in the search bar at the top of the page to find words across all lessons. Tap a result to jump and focus.
Import / Copy / Theme Tap Import, Copy to Clipboard, or the Sun/Moon icons in the header to run these actions.
